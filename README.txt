==========================================
    SYSTEME DE GESTION HOSPITALIER
            Version 1.0.0
==========================================

DESCRIPTION:
Application de gestion hospitalière complète
avec diagnostic médical et archivage.

FONCTIONNALITES:
- Gestion des patients
- Gestion du personnel
- Diagnostic par symptômes
- Archivage des données
- Statistiques

INSTALLATION:
1. Exécutez Setup_Hopital.exe
2. Suivez les instructions à l'écran
3. L'application sera installée dans Program Files

UTILISATION:
- Lancez "Hopital System" depuis le menu Démarrer
- Ou utilisez le raccourci sur le bureau
